# ML-Models

Этот репозиторий содержит решения трех прикладных задач по машинному обучению, охватывающих классификацию, регрессию и временные ряды.

## Содержание
1. [Классификация](#1-классификация)
2. [Регрессия и анализ](#2-регрессия)
3. [Временные ряды](#3-временные-ряды)

---

### 1. Классификация
**Задача:** Сравнение алгоритмов классификации на синтетических данных.
* **Стек:** Scikit-learn, Pipeline, GridSearchCV.
* **Результат:** Оценка моделей по метрике Gini. Использован `Pipeline` для исключения утечки данных.

### 2. Регрессия
**Задача:** Построение ансамблевой модели (Bagging) и анализ качества.
* **Стек:** BaggingRegressor, Statsmodels.
* **Результат:** Проведен анализ мультиколлинеарности (VIF) и анализ остатков модели.

### 3. Временные ряды
**Задача:** Прогнозирование с помощью AR и ETS моделей.
* **Стек:** Statsmodels.
* **Результат:** Сравнение моделей по метрикам MAE и RMSE.
